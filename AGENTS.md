# AI Rules — DREVOS UA Landing

## Project Scope

- This folder is the working project for the Ukraine landing page for DREVOS.
- Keep business-wide memory, analytics, and long-term plans in `/Users/mihail/AI-projects/my-business-ai`.
- Use this folder for landing-page structure, copy, assets, implementation notes, and site-specific decisions.

## How To Work

- Start by reading `memory.md` and `plan.md`.
- Do not ask for clarification over minor wording mistakes if the meaning is clear.
- Give practical recommendations in the format: what to do, where to do it, expected result, how to check.
- Keep the first version simple: landing page first, complex integrations later.
- Do not add API, CRM, webhook, RAG, bot, or deploy complexity unless the user explicitly asks for it.

## Safety

- Do not store API keys, passwords, cookies, tokens, `.env`, payment data, or private client data in this repository.
- Before publishing or pushing to GitHub, check secrets and ask for explicit confirmation.
- Do not delete existing files or rewrite project rules without explicit permission.

## Content Rules

- The landing is for Ukraine first.
- Use DREVOS business facts from `memory.md`, not generic AI or generic ecommerce copy.
- Tie copy and design decisions to trust, lead quality, sales, speed, and conversion.
- Keep tasks and current work in `plan.md`; keep stable facts in `memory.md`.
